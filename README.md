# 港埠雨林甜笋 - 西双版纳高端甜笋企业网站

一个专业的甜笋企业网站，使用 React + Tailwind CSS 构建，展示港埠雨林甜笋的产品、品质保证和购买渠道。专注可生食级高端绿色食材，源自北纬21°西双版纳原始雨林。

## 🚀 快速开始

### 环境要求
- Node.js 14.0 或更高版本
- npm 或 yarn 包管理器

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm start
```

应用将在 `http://localhost:3000` 启动。

### 构建生产版本
```bash
npm run build
```

构建文件将输出到 `build/` 目录。

## 📱 技术特性

- **响应式设计**: 完全适配移动端、平板和桌面设备
- **现代UI设计**: 使用 Tailwind CSS 构建的精美界面
- **配色方案**:
  - 主色: #2E7D32 (绿色)
  - 辅色: #F0B429 (黄色) / #FAF7F0 (米色)
- **无障碍设计**: 按钮对比度≥4.5:1，所有图片含 alt 属性
- **组件化架构**: 使用 React 组件化开发，易于维护

## 📝 自定义配置

### 修改主题色
在 `tailwind.config.js` 中修改颜色配置:

```javascript
theme: {
  extend: {
    colors: {
      primary: '#2E7D32',    // 主色
      secondary: '#F0B429',  // 辅色
      background: '#FAF7F0', // 背景色
    }
  },
}
```

### 替换图片
将组件中的占位图片URL替换为实际产品图片:

```javascript
// 例如在 Hero.jsx 中
style={{
  backgroundImage: "url('your-image-path.jpg')"
}}
```

## 🚀 部署
