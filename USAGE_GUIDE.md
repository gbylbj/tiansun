# 🎋 港埠雨林留言系统使用指南

## 📋 文件说明

### 1. 客户端页面
**文件名**：`customer-contact-form.html`
- **用途**：客户提交留言的表单页面
- **用户**：网站访客/客户
- **功能**：填写姓名、手机号、留言内容并提交

### 2. 管理员页面
**文件名**：`admin-dashboard.html`
- **用途**：管理员查看和下载客户留言数据
- **用户**：网站管理员
- **功能**：查看所有留言、统计数据、导出数据

## 🎯 使用流程

### 客户操作流程
1. **打开** `customer-contact-form.html`
2. **填写**：
   - 姓名（必填）
   - 手机号码（必填，格式验证）
   - 留言内容（必填，至少10个字符）
3. **提交** 点击"提交咨询"按钮
4. **完成** 看到"提交成功"提示即可关闭页面

### 管理员操作流程
1. **打开** `admin-dashboard.html`
2. **查看** 实时统计数据显示（总留言数、今日、本周、本月）
3. **管理**：
   - 查看所有客户留言记录
   - 删除单条记录
   - 导出全部数据（JSON + TXT格式）
   - 导出今日数据
   - 清空所有数据

## 📁 数据导出格式

### JSON 格式
- 文件名：`港埠雨林_全部客户留言_2025-11-05.json`
- 包含：完整的数据结构、时间戳、统计信息
- 适合：数据备份、程序处理

### TXT 格式
- 文件名：`港埠雨林_全部客户留言_2025-11-05.txt`
- 包含：易读的文本格式、格式化显示
- 适合：人工查看、打印记录

## 🔧 数据存储

- **存储位置**：浏览器的 localStorage 中
- **数据结构**：JSON格式
- **持久性**：数据会一直保存，直到手动清理
- **安全性**：只有能访问管理员页面的人才能查看和导出数据

## 💡 使用建议

### 对客户
- 填写完整信息以便我们更好地联系您
- 留言内容详细描述您的需求
- 提交成功后会显示确认信息

## 🚀 部署建议

1. **本地使用**：直接双击 HTML 文件即可使用
2. **网络部署**：上传到服务器即可在线访问
3. **安全建议**：将管理员页面放在需要密码保护的目录中
